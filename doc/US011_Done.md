# US-011: タスク編集画面 完了報告

## 完了定義

- 既存の情報が初期値として設定される
- すべてのフィールドを編集できる
- 変更内容が保存される

## 完了のエビデンス

- `cmd/main.go` にて、タスク編集画面を実装しました。
  - メインメニューでタスクを選択し、"e" キーを押すことで編集画面に遷移できるようにしました。
  - 編集画面では、選択されたタスクの既存データが各入力フィールドに初期値として設定されるようにしました。
  - Enterキーで変更を保存し、Escキーでキャンセルできるようにしました。
  - `app.UpdateTask` を呼び出してタスクの更新処理を行いました。
